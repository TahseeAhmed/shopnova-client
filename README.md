# ShopNova Frontend

A full-featured e-commerce frontend built with React.js and Redux.

## 🛠️ Tech Stack
- React.js 18
- Redux Toolkit
- React Router v6
- Axios
- React Hot Toast
- React Icons
- Custom CSS

## 🚀 Live Website
https://shopnova-client.vercel.app

## ✨ Features
- Beautiful home page with hero section
- Product listing with filters and search
- Product detail page with reviews
- Shopping cart with multi-step checkout
- User authentication (Login/Register)
- Order history
- Admin dashboard
  - Product management (CRUD)
  - Order management
  - Customer management
  - Category management

## ⚙️ Setup Locally

### 1. Install dependencies
npm install

### 2. Create .env file
REACT_APP_API_URL=http://localhost:5000/api

### 3. Start app
npm start
